# 🔮 Glassmorphism Portfolio - Gautham Krishna B

A modern, futuristic portfolio design featuring frosted glass effects, vibrant gradients, and an animated background.

## 🎨 Design Philosophy

Glassmorphism creates a sleek, modern interface with transparency and blur effects. This portfolio uses:
- **Backdrop-filter blur** for frosted glass appearance
- **Animated gradient background** with floating spheres
- **Translucent elements** with subtle borders
- **Vibrant gradients** for accents
- **Mouse-following parallax** for depth
